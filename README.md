# Tavern Games

**Мини‑проект для освоения React и SCSS.**

## Цель

Освоить технологии React и SCSS на практике: создать веб‑приложение с адаптивным интерфейсом, используя компонентный подход и современные методологии стилизации.

## Применяемые технологии

* **React** — библиотека для создания пользовательских интерфейсов.
* **SCSS** — препроцессор CSS с расширенным синтаксисом.

## Применяемый дизайн

**Ссылка на Figma:** https://www.figma.com/site/FNBxNSAf35GNuCKozZ4QsI/Tavern-games?t=IQyMvrHEiapXiUIB-6

### Охваченные экраны

* Главная страница — приветственный экран с информацией о компании.
* Каталог — список игр с фильтрацией и поиском.
* Карточки товаров — отображение отдельной игры с описанием, кнопкой купить и кнопкой возрата к каталогу.

## Необходимые сторонние библиотеки

### `classnames`

**Назначение:** условное объединение CSS‑классов для React‑компонентов. Позволяет динамически формировать `className` на основе пропсов.

**Установка:**
```bash
npm install classnames
# или
yarn add classnames
```
